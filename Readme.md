## In-A-Giffy

This a fun game that allows a group of friends to create hilarious captions on Animated Gifs and see who is the champion of Wit.

## Installation

download or clone the repo and then run:

```
 npm i/yarn add
```

Then to start the app run:

```
meteor run
```
